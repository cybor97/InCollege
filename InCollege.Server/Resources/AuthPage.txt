<ol>
 <li>SignIn</li>
 Авторизирует пользователя. Возвращает токен для доступа к данным. Параметры:
 <ul>
 <li>UserName - имя пользователя</li>
 <li>Password - пароль</li>
 </ul>

 <li>SignUp</li>
 Регистрирует пользователя:
 <ul>
 <li>UserName - имя пользователя</li>
 <li>Password - пароль</li>
 <li>AccountType - тип аккаунта(в числовом представлении)</li>
 <li>BirthDate - дата рождения</li> 
 <li>FullName - ФИО</li> 
 <li>ProfileImage - изображение профиля(функционал не реализован)</li>
 </ul>

 <li>ValidateToken</li>
 Проверяет токен в параметре token. Возвращает пустой ответ с кодом OK в случае, если токен верный и NotAcceptable в противном случае.

 <li>WhoAmI</li>
 Возвращает аккаунт для токена.
</ol>
