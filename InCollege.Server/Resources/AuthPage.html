Авторизация

  1. SignIn
    2. Авторизирует пользователя. Возвращает токен для доступа к данным. Параметры:
    • UserName - имя пользователя
    • Password - пароль
  2. SignUp
    3. Регистрирует пользователя:
    • UserName - имя пользователя
    • Password - пароль
    • AccountType - тип аккаунта(в числовом представлении)
    • BirthDate - дата рождения
    • FullName - ФИО
    • ProfileImage - изображение профиля(функционал не реализован)
  3. ValidateToken
    4. Проверяет токен в параметре token. Возвращает пустой ответ с кодом OK в случае, если токен верный и NotAcceptable в противном случае.
  4. WhoAmI
    5. Возвращает аккаунт для токена.
